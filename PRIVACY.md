# Privacy Policy - WhatsApp Chat Button App

**Last Updated**: January 2026

## Overview

This Privacy Policy describes how the WhatsApp Chat Button app ("the App") collects, uses, and protects information.

## Information We Collect

### App Configuration Data

The App stores the following information in Shopify metafields:
- WhatsApp phone number
- Default message text
- Button position preference
- Enable/disable status

### Customer Data

**The App does NOT collect, store, or process any customer personal data.**

When a customer clicks the WhatsApp button:
- They are redirected to WhatsApp's platform (wa.me)
- No customer information is sent to our servers
- No customer data is stored by the App
- Communication happens directly between the customer and merchant via WhatsApp

## How We Use Information

The configuration data is used solely to:
1. Display the WhatsApp button on the merchant's storefront
2. Configure the button's appearance and behavior
3. Generate the WhatsApp chat link

## Data Storage

- **Configuration data**: Stored in Shopify metafields (managed by Shopify)
- **Sessions**: Stored temporarily in memory (development) or Redis (production)
- **Customer data**: Not stored

## Data Sharing

We do NOT share any data with third parties, except:
- **Shopify**: Configuration data is stored in Shopify's infrastructure
- **WhatsApp**: Customers are redirected to WhatsApp's platform (subject to WhatsApp's Privacy Policy)

## Data Security

- All data transmission uses HTTPS encryption
- OAuth 2.0 authentication via Shopify
- No sensitive customer data is handled
- Shopify's security infrastructure protects stored configuration

## Data Retention

- Configuration data is retained as long as the App is installed
- Upon app uninstallation, configuration data can be deleted (see GDPR section)

## GDPR Compliance

The App complies with GDPR requirements:

### Right to Access
Merchants can view their configuration data in the App admin panel.

### Right to Rectification
Merchants can update their configuration data at any time.

### Right to Erasure
Upon app uninstallation, merchants can request data deletion by contacting support.

### Data Portability
Configuration data can be exported upon request.

## GDPR Webhooks

The App implements the following GDPR webhooks:

1. **customers/data_request**
   - Response: No customer data is stored by this App

2. **customers/redact**
   - Response: No customer data is stored by this App

3. **shop/redact**
   - Action: App configuration data is deleted

## CCPA Compliance

The App does not sell personal information.

California residents have the right to:
- Know what data is collected (see "Information We Collect")
- Request deletion of data
- Opt-out of data selling (not applicable - we don't sell data)

## Children's Privacy

## International Data Transfers
